** Linux and Mac OS X

     virtualenv ~/virtualenvs/diskwalk2
     source ~/virtualenvs/diskwalk2/bin/activate
     pip install diskwalk2
   #+END_SRC

** Windows

   #+BEGIN_SRC sh
     virtualenv C:\virtualenvs\diskwalk2
     C:\virtualenvs\diskwalk2\Scripts\activate
     pip install diskwalk2
   #+END_SRC
